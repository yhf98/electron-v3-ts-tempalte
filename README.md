# Electron-v3-ts-tempalte

> Electron 基于Vue + TS 项目模板


``` bash
# 安装依赖
npm install

# 拷贝lang文件
npm run copy:lang

# 开发阶段启动调试
npm run dev

# 打包构建
npm run build

# 清除缓存
npm cache clean -f
```
